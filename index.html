Installation Guide

1

Include Dependencies

<script src="https://cdnjs.cloudflare.com/ajax/libs/gsap/3.12.5/gsap.min.js"></script>
<script src="https://unpkg.com/split-type@0.3.4/dist/index.umd.js"></script>
2

Add Textify Files

<link rel="stylesheet" href="textify.css">
<script src="textify.js"></script>
3

Initialize Animation

const textify = new Textify('#your-text', {
    style: 'style1',
    duration: 1.5,
    stagger: 0.05,
    ease: 'power2.out'
});
